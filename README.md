# 🏪 StoreMaster

---

StoreMaster is a professional desktop application designed for **small and medium retail businesses**. It provides a complete solution for managing products, suppliers, categories, stock, sales, and reporting — making store management **easy, efficient, and organized**.  

---

## 🌟 Features

- **User Authentication**: Secure login and registration for staff.  
- **Product Management**: Add, edit, delete, search, and display products.  
- **Supplier Management**: Efficiently manage suppliers: add, edit, delete, search, and display.  
- **Category Management**: Organize products into categories.  
- **Stock Management**: Add stock, track stock levels, automatic low-stock alerts, and display stock.  
- **Sales Management**: Select products, specify quantity, auto-calculate total bills, update stock automatically.  
- **Reporting**: Generate detailed product and sales reports.  
- **Professional UI**: Built with Java Swing for a smooth and intuitive desktop experience.  

---

## 🖥 Technology Stack

| Technology | Details |
|------------|---------|
| Language   | Java |
| UI Framework | Swing / AWT |
| Database   | Microsoft SQL Server |
| PDF & Reporting | Java AWT Printing |

---

## 📂 Database Structure

| Table      | Description |
|-----------|------------|
| Users      | Store user credentials and access permissions |
| Products   | Details of all products including price, quantity, supplier, category |
| Suppliers  | Supplier information |
| Categories | Product categories |
| Sales      | Records of all sales transactions |
| Stock      | Records of all the Stocks |
